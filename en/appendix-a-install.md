# Installation
{: .fs-8 }

## Download and Install Rime

1. Download the latest version of Rime (Weasel / 小狼毫) for your operating system from the [official Rime website](https://rime.im/download/) and install it.

## Install the Fengshui Input Method

1. Copy the five Fengshui input method files into the Rime installation folder (`…\Rime\weasel-0.14.3\data`), overwriting the existing files.

   The default installation folder is: `C:\Program Files\Rime\weasel-0.14.3\data`

   The five files:

   - `default.yaml` — default input method settings
   - `weasel.yaml` — input method interface settings
   - `geomancy.dict.yaml` — Fengshui input method dictionary (157,000 words)
   - `geomancy.extended.dict.yaml` — user custom dictionary
   - `geomancy.schema.yaml` — Fengshui input method behavior settings

2. Click "Start" → "Weasel Input Method" (小狼毫输入法) → "Input Method Settings" (输入法设定).

3. Select "Fengshui Form-Phonetic" (风水形音) — click the "中" button to confirm.

   ![Input method settings](/assets/images/install-1.png)

4. Choose an interface style you like, then click the "中" button to confirm.

   ![Interface style](/assets/images/install-2.png)

5. Press <kbd>Ctrl</kbd> + <kbd>`</kbd> (located at the top-left corner) at the same time, then select "Fengshui Form-Phonetic" (风水形音) to finish setting up the input method.

   ![Switch schema](/assets/images/install-3.png)

6. Press <kbd>Ctrl</kbd> + <kbd>Shift</kbd> or <kbd>Ctrl</kbd> + <kbd>Space</kbd> to turn the input method on or off.

   For more detailed usage, see the help on the [official Rime website](https://rime.im/).

## Switch to Pinyin Anytime

If you temporarily cannot recall the input code for a character or word, you can switch at any time to (<kbd>`</kbd> + pinyin) input.

![Pinyin switch](/assets/images/install-4.png)
