# Radical Table
{: .fs-8 }

The Fengshui radical table is built on the **米 (rice) / 人 (person)** glyph structure, combining the *I Ching*, *Yin-Yang and the Five Elements* with English alphabetic pictography, and weaving in birds and beasts, natural elements, body parts and cardinal directions. It covers the radicals of all 8,105 general standard Chinese characters.

![Fengshui radical table (overview)](/assets/images/roots-table.png)

## Left-Hand Radical Table (米 shape)

The left-hand key area uses the "米" glyph as its basic structure, corresponding to the 15 left-hand keys of a QWERTY keyboard (Q W E R T / A S D F G / Z X C V B). Each key gathers a group of radicals that are isomorphic along the stroke directions of the "米" glyph.

![Left-hand radical table](/assets/images/roots-left.png)

## Right-Hand Radical Table (人 shape)

The right-hand key area uses the "人" glyph as its basic structure, corresponding to the 11 right-hand keys (Y U I O P / H J K L / N M) plus the punctuation keys.

![Right-hand radical table](/assets/images/roots-right.png)

---

The Fengshui radical table is easy to learn and remember — you can pick it up in half a day — and decomposition is simple, so you can start typing right away. The input method provides three tiers of simplified codes and three tiers of candidate words: you can commit with a short code or with the full code, with precise control over every character and every word. Once you become proficient with the Fengshui Input Method, it feels like driving an F1 race car tearing down a track — that sense of "doing as you please," of "total control," of "making others envious," is something no other input method can match.
