# 输入法安装
{: .fs-8 }

## 下载并安装 Rime 输入法

1. 在 [Rime 官网](https://rime.im/download/) 下载适用于自己电脑系统的最新版 Rime（小狼毫）输入法，并安装。

## 安装风水输入法

1. 将风水输入法的五个文件复制到 Rime 安装文件夹（`…\Rime\weasel-0.14.3\data`）中并覆盖原有文件。

   默认安装文件夹为：`C:\Program Files\Rime\weasel-0.14.3\data`

   五个文件：

   - `default.yaml` —— 输入法默认设置文件
   - `weasel.yaml` —— 输入法界面设置文件
   - `geomancy.dict.yaml` —— 风水输入法词库文件（15.7 万词条）
   - `geomancy.extended.dict.yaml` —— 用户自定义词库文件
   - `geomancy.schema.yaml` —— 风水输入法功能设置文件

2. 点击「开始」——「小狼毫输入法」——「输入法设定」。

3. 选择「风水形音」——点击「中」确认。

   ![输入法设定](/assets/images/install-1.png)

4. 选择一个喜欢的界面风格，点击「中」确认。

   ![界面风格](/assets/images/install-2.png)

5. 同时点击 <kbd>Ctrl</kbd> + <kbd>`</kbd>（位于左上角），选择「风水形音」，完成输入法设定。

   ![切换方案](/assets/images/install-3.png)

6. 按 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> 或 <kbd>Ctrl</kbd> + <kbd>空格</kbd>，打开或关闭输入法。

   更详细的使用方法可以查看 [Rime 官网](https://rime.im/) 帮助。

## 随时切换为拼音

若临时想不起来某个字或词组的输入编码，可以随时切换为（<kbd>`</kbd> + 拼音）的方式输入。

![拼音切换](/assets/images/install-4.png)
